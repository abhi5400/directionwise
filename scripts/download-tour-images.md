## Image Specifications

- **Format**: JPEG (JPG)
- **Dimensions**: 1200x675px (16:9 aspect ratio) recommended
- **File Size**: Under 500KB per image
- **Quality**: High resolution, professional photography

## Download Steps

1. Visit the recommended links above
2. Search for the specific tour images
3. Filter by "Free" or "Royalty Free" license
4. Download high-resolution versions
5. Rename files exactly as specified above
6. Place in `assets/images/tours/` directory
7. Run image conversion script to create WebP versions

## Quick Download Script

After downloading images, you can use the conversion script:

```bash
# Place downloaded images in assets/images/source/
# Then run:
npm install
npm run convert-images
```
